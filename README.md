# C/S study - database

## 소개

데이터베이스 이론을 공부하고 정리한 자료를 저장하는 공간입니다. 또한 이곳에 코딩악마 팀의 데이터베이스 스터디에서 매주 진행하는 발표와 질의응답 내용을 개인적으로 기록합니다.

문서는 토픽 별로 정리되어 있고, 각 토픽 별 간단한 퀴즈도 포함되어 있습니다.
